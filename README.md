# LinkageTableViewDemo
一款可多级联动的楼盘表demo，可上下左右联动，可进行手势捏合进行缩放，支持快速定位，包含表格制作，横竖向表格展示。大部分功能主要实现原理为UITableView或UICollectionView的灵活嵌套及运用，以此来保证横竖向滑动时皆能复用，避免数据量大时滑动卡顿或卡死，具体细节可下载源码进行查看。
  
* 注：尽量用真机调试，模拟器部分页面有可能会显示失常！
