# ZHLinkageChartView
一款可多级联动的表格，可上下左右联动。主要实现原理为UICollectionView的灵活嵌套，以此来保证横竖向滑动时皆能复用，避免数据量大时滑动卡顿或卡死，具体细节可下载源码进行查看。
  
#### 支持功能
* 支持多向联动
* 支持行列任意拓展
* 支持表格上下任意合并
